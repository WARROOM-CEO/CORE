# WARROOM-CEO/CORE — คลังปลั๊กอิน Claude Code แบบ Localized

**ปลั๊กอิน Cowork ภาษาไทยสำหรับทีมงานและองค์กร**  
ปลั๊กอินที่แปลและปรับแต่งเป็นภาษาไทยทั้งหมด พร้อมใช้งานใน Claude Cowork และ Claude Code

---

## เกี่ยวกับคลังนี้

คลัง **WARROOM-CEO/CORE** เป็นคอลเลกชันของปลั๊กอิน Claude Code ที่ถูกแปลและปรับแต่งให้ใช้งานได้อย่างสมบูรณ์ในภาษาไทย  
แต่ละปลั๊กอินได้รับการปรับแต่งเฉพาะสำหรับทีมงานและองค์กร เพื่อให้การทำงานเป็นไปอย่างราบรื่นและมีประสิทธิภาพ

### คุณสมบัติเด่น
- ✅ **รองรับภาษาไทยเต็มรูปแบบ** — ทุกส่วนที่ผู้ใช้เห็นเป็นภาษาไทย
- ✅ **ปรับแต่งสำหรับองค์กรไทย** — เชื่อมต่อกับเครื่องมือที่ใช้จริงในองค์กร
- ✅ **โครงสร้างสองภาษา (Bilingual)** — โค้ดหลักภาษาอังกฤษ, UI ภาษาไทย
- ✅ **รองรับ MCP Servers** — เชื่อมต่อกับบริการภายนอกได้ทันที
- ✅ **ทดสอบแล้วใน Cowork** — ทำงานได้สมบูรณ์ในแอปเดสก์ท็อป

## การติดตั้ง

### ผ่าน Claude Cowork (แนะนำ)
1. เปิด Claude Cowork Desktop App
2. พิมพ์คำสั่ง: `/plugin marketplace add WARROOM-CEO/CORE`
3. เลือกปลั๊กอินที่ต้องการติดตั้งจากรายการ

### ติดตั้งแบบ manual
1. Clone repository นี้:
```bash
git clone https://github.com/WARROOM-CEO/CORE.git
```
2. ใน Claude Cowork/Code:
   - ไปที่ Settings → Plugins
   - กด "Add Local Plugin"
   - เลือกโฟลเดอร์ปลั๊กอินที่ต้องการจาก directory `plugins/`

ทุกปลั๊กอินในคลังนี้ใช้โครงสร้างสองภาษา ตามมาตรฐานของ WARROOM-CEO:

| ไฟล์ | ภาษา | วัตถุประสงค์ |
|------|-------|--------------|
| `SKILL.md` | อังกฤษ + ไทย (ส่วนสื่อกับผู้ใช้) | ลด Token — เนื้อหาหลักอังกฤษ, output/คำถามเป็นไทย |
| `SKILL-TH.md` | ไทยทั้งหมด | อ่านทำความเข้าใจ Skill |
| `README.md` | ไทย | คำอธิบาย Plugin + License section |
| `LICENSE` | อังกฤษ | Apache 2.0 ฉบับเต็ม |
| `references/*.md` | อังกฤษ | เอกสารอ้างอิงภายใน |
| `references/*-TH.md` | ไทยทั้งหมด | คำแปลเอกสารอ้างอิงสำหรับอ่านทำความเข้าใจ |
| `agents/*.md` | อังกฤษ | คำสั่ง subagent ภายใน |
| `agents/*-TH.md` | ไทยทั้งหมด | คำแปลคำสั่ง subagent สำหรับอ่านทำความเข้าใจ |
| `commands/*.md` | อังกฤษ + ไทย (ส่วนสื่อกับผู้ใช้) | Slash command รูปแบบ Legacy |
