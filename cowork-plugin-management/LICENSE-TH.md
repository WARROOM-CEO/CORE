## สัญญาอนุญาต Apache เวอร์ชัน 2.0 (มกราคม 2547)

สงวนลิขสิทธิ์ ตามเงื่อนไขการใช้งาน การทำซ้ำ และการแจกจ่ายงานนี้ รวมถึงงานดัดแปลงทั้งใน Source Form และ Object Form ภายใต้เงื่อนไขต่อไปนี้

---

## บทนิยาม

| คำศัพท์ | ความหมาย |
|---------|----------|
| **สัญญาอนุญาต** | ข้อกำหนดและเงื่อนไขการใช้งาน การทำซ้ำ และการแจกจ่ายตามเอกสารนี้ |
| **ผู้อนุญาต** | เจ้าของลิขสิทธิ์หรือนิติบุคคลที่ได้รับมอบอำนาจในการให้สิทธิ์ |
| **คุณ** | บุคคลธรรมดาหรือนิติบุคคลที่ใช้สิทธิ์ภายใต้สัญญาอนุญาตนี้ |
| **Source Form** | รูปแบบที่เหมาะสำหรับการแก้ไข เช่น ซอร์สโค้ด เอกสาร และไฟล์กำหนดค่า |
| **Object Form** | รูปแบบที่ได้จากการแปลง Source Form เช่น ไฟล์ที่คอมไพล์แล้ว |
| **งาน (Work)** | งานสร้างสรรค์ที่เผยแพร่ภายใต้สัญญาอนุญาตนี้ |
| **งานดัดแปลง** | งานที่อิงหรือดัดแปลงจากงานต้นฉบับ โดยที่การเพิ่มเติมถือเป็นงานสร้างสรรค์ใหม่ |
| **การมีส่วนร่วม** | งานที่ส่งให้ผู้อนุญาตเพื่อรวมเข้าในงาน |
| **ผู้มีส่วนร่วม** | ผู้อนุญาตและนิติบุคคลที่ส่งการมีส่วนร่วมเข้ามา |

## การให้สิทธิ์

### สิทธิ์ลิขสิทธิ์

ผู้มีส่วนร่วมแต่ละรายให้สิทธิ์ **ถาวร ทั่วโลก ไม่ผูกขาด ไม่มีค่าธรรมเนียม ปลอดค่าลิขสิทธิ์** ในการ:

- ทำซ้ำงาน
- จัดเตรียมงานดัดแปลง
- แสดงและดำเนินการต่อสาธารณะ
- ให้สิทธิ์ช่วง
- แจกจ่ายทั้งใน Source Form และ Object Form

### สิทธิ์สิทธิบัตร

ผู้มีส่วนร่วมแต่ละรายให้สิทธิ์สิทธิบัตรอย่างถาวร ทั่วโลก ไม่ผูกขาด ไม่มีค่าธรรมเนียม ในการสร้าง ใช้งาน เสนอขาย ขาย นำเข้า และถ่ายโอนงาน

## ข้อจำกัดและการปฏิเสธความรับผิด

### เครื่องหมายการค้า

สัญญาอนุญาตนี้ **ไม่ได้ให้สิทธิ์** ในการใช้ชื่อทางการค้า เครื่องหมายการค้า เครื่องหมายบริการ หรือชื่อผลิตภัณฑ์ของผู้อนุญาต

### การปฏิเสธการรับประกัน

ผู้อนุญาตให้งานนี้ **"ตามสภาพที่เป็น"** โดย **ไม่มีการรับประกันใด ๆ** ทั้งโดยชัดแจ้งหรือโดยนัย รวมถึงความเหมาะสมในทางการค้าหรือความเหมาะสมสำหรับวัตถุประสงค์เฉพาะ

### การจำกัดความรับผิด

ผู้มีส่วนร่วม **จะไม่รับผิดต่อความเสียหายใด ๆ** รวมถึงความเสียหายโดยตรง โดยอ้อม พิเศษ อุบัติเหตุ หรือผลที่ตามมา อันเกิดจากการใช้หรือไม่สามารถใช้งานได้
