# frontend-design

**สกิล Frontend Design สำหรับ Cowork**

สกิลนี้ช่วยสร้างอินเทอร์เฟซ Frontend ที่โดดเด่นและพร้อม production ด้วยคุณภาพการออกแบบสูง หลีกเลี่ยงรูปแบบ AI ทั่วไปที่ไม่มีเอกลักษณ์ Claude จะสื่อสารและอธิบายแนวคิดการออกแบบกับผู้ใช้เป็น**ภาษาไทย**ตลอดการใช้งาน

---

## Skills ที่มีในปลั๊กอินนี้

### `/frontend-design` — ออกแบบและสร้าง Frontend

ใช้เมื่อต้องการสร้าง web components, หน้าเว็บ, หรือแอปพลิเคชัน

**เรียกใช้ด้วย:**
- "สร้างหน้าเว็บ"
- "ทำ component นี้ให้หน่อย"
- "ออกแบบ UI สำหรับ..."
- "build a landing page" / "create a dashboard"

**สิ่งที่ทำได้:**
- สร้าง web components, หน้าเว็บ, และแอปพลิเคชัน (HTML/CSS/JS, React, Vue ฯลฯ)
- ออกแบบ UI/UX ที่มีเอกลักษณ์และหลีกเลี่ยงสไตล์ AI ทั่วไป
- เลือก typography, สี, และ animation ที่สอดคล้องกับบริบทของโปรเจกต์
- ส่งมอบโค้ดที่พร้อม production และใช้งานได้จริง

## การติดตั้ง

1. ดับเบิลคลิกไฟล์ `frontend-design.plugin` ใน Cowork
2. กด **ยืนยันการติดตั้ง** ในหน้าต่าง preview
3. เรียกใช้ได้ทันทีด้วยคำสั่งข้างต้น
