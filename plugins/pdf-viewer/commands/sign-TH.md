# เซ็นชื่อ PDF (`/pdf-viewer:sign`)

**วางรูปลายเซ็นหรืออักษรย่อบน PDF**

---

## คำสั่งนี้ทำอะไร?

เพิ่มลายเซ็นหรืออักษรย่อแบบภาพลงในเอกสาร โดยใช้ annotation ประเภท image

> **ข้อสำคัญ:** คำสั่งนี้วาง **รูปลายเซ็น** บนหน้า เท่านั้น — **ไม่ใช่** ลายเซ็นดิจิทัลที่รับรองหรือเข้ารหัสด้วยเทคโนโลยี cryptographic สำหรับ e-signature ที่มีผลทางกฎหมาย ให้ใช้บริการ signing โดยเฉพาะ

---

## วิธีใช้

```
/pdf-viewer:sign
/pdf-viewer:sign [path ของ PDF] [path ของรูปลายเซ็น]
```

---

## ขั้นตอนการทำงาน

1. **รับรูปลายเซ็น** — Claude จะถามหา path ไฟล์ PNG/JPG ลายเซ็นหรืออักษรย่อ ถ้ายังไม่มีจะแนะนำให้สร้างและบันทึกไว้ก่อน

2. **เปิด PDF** — เปิดในตัวแสดงผล ตรวจสอบ `formFields` เพื่อดูว่ามีช่องลายเซ็นพร้อม coordinates หรือไม่

3. **หาตำแหน่ง** — ถ้ามีช่องลายเซ็น จะใช้ coordinates ของช่องนั้น ถ้าไม่มีจะถาม "หน้าไหน และตำแหน่งใดบนหน้า? เช่น มุมล่างขวาหน้า 3"

4. **วางลายเซ็น** — เพิ่ม annotation ประเภท image ที่ตำแหน่งที่ต้องการ

5. **ยืนยัน** — screenshot หน้านั้นและแสดงให้คุณดู ปรับตำแหน่งได้ถ้าต้องการ

6. **อักษรย่อทุกหน้า** — วาง annotation image หนึ่งชิ้นต่อหน้าในการเรียกครั้งเดียว

---

## เคล็ดลับ

- `imageUrl` รับ local file path หรือ HTTPS URL (ไม่รับ data: URI)
- คุณสามารถลากวางรูปลายเซ็นโดยตรงลงใน viewer ได้เช่นกัน
- จุดเริ่มต้น coordinates คือมุมบนซ้าย; ลายเซ็นมุมล่างขวาบนกระดาษ US Letter อยู่แถวๆ `x: 400, y: 700`
- ใช้คู่กับ `/pdf-viewer:fill-form` สำหรับ workflow แบบฟอร์มครบ
