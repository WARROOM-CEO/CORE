# คู่มือการเชื่อมต่อเครื่องมือ

## วิธีที่ปลั๊กอินใช้อ้างอิงเครื่องมือ

ไฟล์ต่างๆ ในปลั๊กอินใช้ `~~หมวดหมู่` เป็นตัวแทนชั่วคราวสำหรับเครื่องมือที่คุณเชื่อมต่อในหมวดนั้น ตัวอย่างเช่น `~~ระบบติดตามงาน` อาจหมายถึง Asana, Linear, Jira หรือระบบติดตามงานอื่นที่มี MCP server

ปลั๊กอินออกแบบมาให้ **ไม่ผูกติดกับเครื่องมือเฉพาะเจาะจง** — อธิบายกระบวนการทำงานในแง่ของหมวดหมู่ (ระบบแชท ระบบติดตามงาน ฐานความรู้ ฯลฯ) ไม่ใช่ชื่อผลิตภัณฑ์ ไฟล์ `.mcp.json` จะกำหนดค่า MCP server เฉพาะไว้ล่วงหน้า แต่ MCP server ใดก็ตามในหมวดหมู่นั้นสามารถใช้งานได้
