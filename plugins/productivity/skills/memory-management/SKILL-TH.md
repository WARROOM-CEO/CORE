# สกิล: การจัดการหน่วยความจำ

**ระบบหน่วยความจำสองชั้นที่ทำให้ Claude เป็นเพื่อนร่วมงานที่เข้าใจภาษาภายในองค์กร**

---

## สกิลนี้ทำอะไร?

`การจัดการหน่วยความจำ` สอนให้ Claude เข้าใจคำย่อ คำศัพท์เฉพาะ ชื่อเล่น และศัพท์เฉพาะภายในองค์กรของคุณ — เพื่อให้สามารถตอบสนองคำขอได้เหมือนเพื่อนร่วมงาน ไม่ใช่แชทบอททั่วไป

**ตัวอย่าง:**
```
คุณพิมพ์:  "ให้ todd ทำรายงานสรุปสถานะโครงการสำหรับ oracle ด้วย"
Claude เข้าใจ: "ขอให้ Todd Martinez (หัวหน้าฝ่ายการเงิน) จัดทำรายงานสรุปสถานะโครงการ
               สำหรับดีล Oracle Systems ($2.3M, ปิดในไตรมาส 2)"
```

โดยไม่ต้องถามซ้ำ ไม่ต้องอธิบายซ้ำ

---

## โครงสร้างหน่วยความจำสองชั้น

```
CLAUDE.md          ← แคชร้อน (~30 คน คำศัพท์ที่ใช้บ่อย)
memory/
  glossary.md      ← คู่มือถอดรหัสครบทุกคำ
  people/          ← ข้อมูลส่วนตัวแต่ละคนอย่างละเอียด
  projects/        ← รายละเอียดโครงการ
  context/         ← ทีม เครื่องมือ กระบวนการทำงานขององค์กร
```

**CLAUDE.md (แคชร้อน):**
- คน ~30 คนที่ทำงานด้วยบ่อยที่สุด
- คำย่อ/คำศัพท์ที่ใช้บ่อย ~30 คำ
- โครงการที่กำลังดำเนินการอยู่ (5-15 รายการ)
- ความชอบส่วนตัว
- เป้าหมาย: ครอบคลุม 90% ของการถอดรหัสในชีวิตประจำวัน

**memory/ (ที่เก็บข้อมูลเต็ม):**
- เก็บทุกคน ทุกคำ ทุกโครงการที่ไม่ได้อยู่ใน CLAUDE.md
- เข้าถึงเมื่อต้องการรายละเอียดเพิ่มเติม

---

## ลำดับการค้นหา (กระบวนการทำงาน)

```
1. ตรวจสอบ CLAUDE.md ก่อน → ครอบคลุม 90% ของกรณี
2. ค้นหาไฟล์ memory/glossary.md → ถ้าไม่พบในแคชร้อน
3. ค้นหาโฟลเดอร์ memory/people/, projects/ → สำหรับรายละเอียดครบถ้วน
4. สอบถามผู้ใช้ → ถ้าไม่รู้จักคำ → จดจำไว้
```

## รูปแบบไฟล์

### CLAUDE.md (หน่วยความจำทำงาน, ~50-80 บรรทัด)

```markdown
# หน่วยความจำ

## ตัวฉัน
[ชื่อ], [ตำแหน่ง] ใน [ทีม].

## บุคคล
| ใคร | บทบาท |
|-----|------|
| **Todd** | Todd Martinez, หัวหน้าฝ่ายการเงิน |
| **Sarah** | Sarah Chen, วิศวกร (แพลตฟอร์ม) |

## คำศัพท์
| คำศัพท์ | ความหมาย |
|------|---------|
| PSR | รายงานสรุปสถานะโครงการ |
| P0 | งานด่วนที่สุด หยุดทุกอย่าง |

## โครงการ
| ชื่อ | รายละเอียด |
|------|------|
| **Phoenix** | การโยกย้ายฐานข้อมูล เปิดตัวไตรมาส 2 |

## ความชอบส่วนตัว
- การประชุมไม่เกิน 25 นาที
- ให้ความสำคัญกับการสื่อสารแบบไม่เรียลไทม์ Slack ดีกว่าอีเมล
```

## กฎการจัดการหน่วยความจำ

**เลื่อนขึ้นไปยัง CLAUDE.md เมื่อ:** ใช้บ่อย / เป็นงานที่กำลังทำอยู่

**เลื่อนลงมาอยู่ใน memory/ เมื่อ:** โครงการเสร็จแล้ว / บุคคลนั้นไม่ได้ทำงานด้วยบ่อยแล้ว / คำนั้นใช้น้อยลง
