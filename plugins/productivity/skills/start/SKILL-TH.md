# สกิล: เริ่มต้นระบบ (`/productivity:start`)

**เริ่มต้นระบบประสิทธิภาพการทำงานและเปิดแผงควบคุม**

---

## ใช้เมื่อไหร่?

- ตั้งค่าปลั๊กอินครั้งแรก
- เริ่มต้นหน่วยความจำทำงานจากรายการงานที่มีอยู่
- ถอดรหัสคำย่อ (ชื่อเล่น คำย่อ ชื่อโครงการ) ในการทำงานของคุณ

---

## สิ่งที่ Claude จะทำ

### 1. ตรวจสอบว่ามีอะไรอยู่แล้ว
ค้นหาในโฟลเดอร์ทำงาน:
- `TASKS.md` — รายการงาน
- `CLAUDE.md` — หน่วยความจำทำงาน
- `memory/` — โฟลเดอร์ฐานข้อมูลเชิงลึก
- `dashboard.html` — ส่วนติดต่อผู้ใช้สำหรับดูงานและข้อมูลอ้างอิง

### 2. สร้างสิ่งที่ขาดหายไป
- ถ้าไม่มี `TASKS.md` → สร้างพร้อมเทมเพลตมาตรฐาน
- ถ้าไม่มี `dashboard.html` → คัดลอกจากปลั๊กอิน
- ถ้าไม่มี `CLAUDE.md` และ `memory/` → เริ่มต้นระบบหน่วยความจำทำงาน (ดูด้านล่าง)

### 3. เปิดแผงควบคุม
Claude จะแจ้งให้ผู้ใช้เปิด `dashboard.html` จากเบราว์เซอร์ไฟล์ด้วยตัวเอง (เนื่องจาก Claude ทำงานในเครื่องเสมือน ไม่สามารถเปิดเบราว์เซอร์โดยตรงได้)

### 4. เริ่มต้นระบบหน่วยความจำทำงาน (ครั้งแรกเท่านั้น)
ถ้ายังไม่มี `CLAUDE.md` และ `memory/` Claude จะสอบถาม:

```
คุณเก็บรายการสิ่งที่ต้องทำไว้ที่ไหน?
- ไฟล์ในเครื่อง (เช่น TASKS.md, todo.txt)
- แอปพลิเคชัน (เช่น Asana, Linear, Jira, Notion)
- ไฟล์บันทึกย่อ

Claude จะใช้งานของคุณเพื่อเรียนรู้ภาษาภายในองค์กร
```

จากนั้น Claude จะถอดรหัสแต่ละงานทีละรายการ เช่น:
```
งาน: "ส่งรายงานสรุปสถานะโครงการให้ Todd เรื่องอุปสรรค Phoenix"

คำที่ต้องการความเข้าใจ:
1. รายงานสรุปสถานะโครงการ — ย่อมาจากอะไร?
2. Todd — ใคร? (ชื่อเต็ม ตำแหน่ง)
3. Phoenix — ชื่อโครงการใช่ไหม? เกี่ยวกับอะไร?
```

### 5. สร้างไฟล์ข้อมูลอ้างอิง
จากข้อมูลที่รวบรวมได้ Claude จะสร้าง:
- `CLAUDE.md` — หน่วยความจำทำงาน (~50-80 บรรทัด)
- `memory/glossary.md` — คู่มือถอดรหัสครบทุกคำ
- `memory/people/{name}.md` — ข้อมูลส่วนตัวแต่ละคน
- `memory/projects/{name}.md` — รายละเอียดโครงการ
- `memory/context/company.md` — ทีม เครื่องมือ กระบวนการทำงาน

### 6. รายงานผล
```
ระบบประสิทธิภาพการทำงานพร้อมใช้งาน:
- งาน: TASKS.md (X รายการ)
- ข้อมูลอ้างอิง: X คน X คำ X โครงการ
- แผงควบคุม: เปิดได้จากเบราว์เซอร์ไฟล์

ใช้ /productivity:update เพื่ออัปเดต (เพิ่ม --comprehensive เพื่อสแกนเชิงลึก)
```

## หมายเหตุ

- ถ้าระบบหน่วยความจำทำงานเริ่มต้นแล้ว `/start` จะเปิดแค่แผงควบคุม
- ชื่อเล่นสำคัญมาก — ต้องจับว่าคนนั้นถูกเรียกว่าอะไรในชีวิตจริง
- ถ้าแหล่งข้อมูลใดไม่พร้อม ให้ข้ามไปและบันทึกช่องว่างไว้
