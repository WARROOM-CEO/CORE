# การคัดกรองข้อตกลงความลับ — คัดกรองข้อตกลงความลับอย่างรวดเร็ว

**คัดกรองข้อตกลงความลับที่เข้ามาอย่างรวดเร็วและจำแนกเป็นสีเขียว สีเหลือง หรือสีแดง**

---

## สกิลนี้ทำอะไร?

คัดกรองข้อตกลงความลับที่เข้ามาตามเกณฑ์มาตรฐาน จำแนกเป็น 3 ระดับเพื่อกำหนดเส้นทางการดำเนินการที่เหมาะสม: อนุมัติตามมาตรฐาน ทบทวนโดยทนายความ หรือต้องได้รับการตรวจสอบทางกฎหมายอย่างสมบูรณ์

> **หมายเหตุสำคัญ**: ช่วยในการทำงานด้านกฎหมายแต่ไม่ใช่การให้คำแนะนำทางกฎหมาย การวิเคราะห์ทั้งหมดควรได้รับการตรวจสอบโดยผู้เชี่ยวชาญด้านกฎหมาย

---

## เกณฑ์การคัดกรอง 10 ข้อ

| เกณฑ์ | สิ่งที่ตรวจสอบ |
|-------|--------------|
| **1. โครงสร้างข้อตกลง** | ประเภทข้อตกลงความลับ เหมาะกับบริบทหรือไม่ |
| **2. คำจำกัดความของข้อมูลความลับ** | ขอบเขตสมเหตุสมผล มีข้อยกเว้นที่จำเป็น |
| **3. ภาระผูกพันของฝ่ายรับข้อมูล** | มาตรฐานการดูแล ข้อจำกัดการใช้งาน |
| **4. ข้อยกเว้นมาตรฐาน** | ความรู้สาธารณะ การครอบครองมาก่อน การพัฒนาอิสระ การได้รับจากบุคคลที่สาม คำสั่งศาล |
| **5. การเปิดเผยที่ได้รับอนุญาต** | พนักงาน ที่ปรึกษา บริษัทในเครือ หน่วยงานกำกับดูแล |
| **6. ระยะเวลาและความยาว** | ระยะเวลาสมเหตุสมผล (1-3 ปีมาตรฐาน) ไม่ถาวร |
| **7. การส่งคืนและการทำลาย** | มีข้อยกเว้นการเก็บรักษาสำหรับกฎหมาย/การปฏิบัติตามกฎระเบียบ |
| **8. การเยียวยา** | การเยียวยาโดยคำสั่งศาลเป็นมาตรฐาน ไม่มีค่าเสียหายที่กำหนดล่วงหน้า |
| **9. ข้อความที่มีปัญหา** | ไม่มีข้อห้ามเชิญชวนพนักงาน ข้อห้ามแข่งขัน สิทธิพิเศษเฉพาะ สิทธิ์ในทรัพย์สินทางปัญญา |
| **10. กฎหมายที่ใช้บังคับ** | เขตอำนาจที่เหมาะสม ไม่มีการอนุญาโตตุลาการบังคับ |

## 3 ระดับการจำแนก

### 🟢 สีเขียว — อนุมัติตามมาตรฐาน

**ทุกข้อต่อไปนี้ต้องเป็นจริง:**
- ข้อตกลงความลับเป็นแบบทั้งสองฝ่าย (หรือแบบฝ่ายเดียวในทิศทางที่เหมาะสม)
- มีข้อยกเว้นมาตรฐานครบทั้ง 5 ข้อ
- ระยะเวลอยู่ในช่วงมาตรฐาน (1-3 ปี ระยะเวลาการคงอยู่ 2-5 ปี)
- ไม่มีข้อห้ามเชิญชวนพนักงาน ข้อห้ามแข่งขัน สิทธิพิเศษเฉพาะ
- ไม่มีข้อความส่วนที่เหลือหรือมีแต่ขอบเขตแคบ
- กฎหมายที่ใช้บังคับอยู่ในเขตอำนาจที่เหมาะสม

**เส้นทางการดำเนินการ:** อนุมัติตามการมอบอำนาจมาตรฐานได้ทันที

### 🟡 สีเหลือง — ต้องทบทวนโดยทนายความ

**มีหนึ่งรายการหรือมากกว่าต่อไปนี้ แต่ข้อตกลงความลับไม่มีปัญหาพื้นฐาน:**
- คำจำกัดความของข้อมูลความลับกว้างกว่าที่ต้องการแต่ยังพอรับได้
- ระยะเวลายาวกว่ามาตรฐานแต่อยู่ในช่วงที่พบได้ทั่วไป
- ขาดข้อยกเว้นมาตรฐานหนึ่งรายการที่เพิ่มได้ง่าย
- ข้อความส่วนที่เหลือแคบ (ความจำตามธรรมชาติเท่านั้น)
- กฎหมายที่ใช้บังคับอยู่ในเขตอำนาจที่พอรับได้แต่ไม่ใช่ที่ต้องการ

**เส้นทางการดำเนินการ:** ระบุประเด็นเฉพาะให้ทนายความทบทวน — น่าจะแก้ไขได้ในการทบทวนครั้งเดียว

### 🔴 สีแดง — มีปัญหาสำคัญ

**มีหนึ่งรายการหรือมากกว่าต่อไปนี้:**
- แบบฝ่ายเดียวในกรณีที่ต้องการแบบทั้งสองฝ่าย (หรือผิดทิศทาง)
- ขาดข้อยกเว้นที่สำคัญ (โดยเฉพาะการพัฒนาอิสระหรือคำสั่งศาล)
- มีข้อห้ามเชิญชวนพนักงานหรือข้อห้ามแข่งขันซ่อนอยู่
- มีสิทธิพิเศษเฉพาะหรือข้อห้ามการดำเนินการ
- ระยะเวลาระดับไม่สมเหตุสมผล (10+ ปีหรือถาวร)
- คำจำกัดความที่กว้างเกินไปอาจครอบคลุมข้อมูลสาธารณะ
- การโอนสิทธิ์ในทรัพย์สินทางปัญญาหรือการให้สิทธิ์ใช้งานซ่อนอยู่ในข้อตกลงความลับ
- ค่าเสียหายที่กำหนดล่วงหน้าหรือข้อลงโทษ

**เส้นทางการดำเนินการ:** ห้ามลงนาม ต้องผ่านการทบทวนทางกฎหมายอย่างสมบูรณ์และการเจรจาหรือเสนอข้อตกลงใหม่
