# การตรวจสอบสัญญา — ตรวจสอบสัญญาตามแนวทางการเจรจา

**ตรวจสอบสัญญาตามแนวทางการเจรจาต่อรองขององค์กร — ระบุข้อแตกต่าง สร้างข้อเสนอแก้ไข วิเคราะห์ผลกระทบทางธุรกิจ**

---

## สกิลนี้ทำอะไร?

ตรวจสอบสัญญาตามแนวทางการเจรจาต่อรองขององค์กร วิเคราะห์แต่ละข้อความ ระบุข้อที่แตกต่างจากมาตรฐาน สร้างข้อเสนอแก้ไขที่เจาะจง และให้การวิเคราะห์ผลกระทบทางธุรกิจ

> **หมายเหตุสำคัญ**: ช่วยในการทำงานด้านกฎหมายแต่ไม่ใช่การให้คำแนะนำทางกฎหมาย การวิเคราะห์ควรได้รับการตรวจสอบโดยผู้เชี่ยวชาญด้านกฎหมายที่มีคุณวุฒิ

## ระบบจำแนก 3 ระดับ

### 🟢 สีเขียว — ยอมรับได้
ข้อความในสัญญาตรงกับหรือดีกว่าตำแหน่งมาตรฐานขององค์กร
**การดำเนินการ:** บันทึกไว้เพื่อรับรู้ ไม่จำเป็นต้องเจรจา

### 🟡 สีเหลือง — ควรเจรจา
ข้อความในสัญญาออกนอกตำแหน่งมาตรฐานแต่อยู่ในช่วงที่สามารถเจรจาได้
**การดำเนินการ:** สร้างข้อเสนอแก้ไขที่เจาะจง ให้ทางเลือกสำรอง ประเมินผลกระทบทางธุรกิจ

### 🔴 สีแดง — ควรแจ้งเตือนระดับสูง
ข้อความในสัญญาออกนอกช่วงที่ยอมรับได้หรือเป็นเงื่อนไขที่ต้องแจ้งเตือนระดับสูง
**การดำเนินการ:** อธิบายความเสี่ยงเฉพาะเจาะจง ให้ทางเลือกตามมาตรฐานตลาด ประเมินความเสี่ยงที่อาจเกิดขึ้น แนะนำเส้นทางการแจ้งเตือน

## ข้อความที่วิเคราะห์ (อย่างน้อย)

| ข้อความ | ประเด็นหลักที่ตรวจสอบ |
|--------|-----------------|
| **ข้อจำกัดความรับผิด** | จำนวนเงินสูงสุด ข้อยกเว้น ความรับผิดทั้งสองฝ่าย vs ฝ่ายเดียว ความเสียหายที่ตามมา |
| **การชดใช้ค่าเสียหาย** | ขอบเขต ความรับผิดทั้งสองฝ่าย vs ฝ่ายเดียว จำนวนเงินสูงสุด การละเมิดสิทธิ์ในทรัพย์สินทางปัญญา การละเมิดข้อมูล |
| **ความเป็นเจ้าของทรัพย์สินทางปัญญา** | ทรัพย์สินทางปัญญาที่มีอยู่ก่อน ทรัพย์สินทางปัญญาที่พัฒนาขึ้นใหม่ งานที่จ้างทำ ใบอนุญาตที่ให้ |
| **การคุ้มครองข้อมูล** | ความต้องการสัญญาการคุ้มครองข้อมูล ผู้ให้บริการย่อย การแจ้งเตือนเมื่อมีการละเมิด การโอนข้ามพรมแดน |
| **ข้อตกลงความลับ** | ขอบเขต ระยะเวลา ข้อยกเว้น การส่งคืน/การทำลาย |
| **ระยะเวลาและการยกเลิกสัญญา** | ระยะเวลา ต่ออายุ การยกเลิกตามความสะดวก การทยอยสิ้นสุดสัญญา |
| **กฎหมายที่ใช้บังคับ** | อำนาจศาล สถานที่ฟ้องร้อง การอนุญาโตตุลาการ vs การดำเนินคดี |
| **ประกันภัย** | ความต้องการความคุ้มครอง จำนวนเงินขั้นต่ำ |
| **การโอนสิทธิ** | ความต้องการความยินยอม การเปลี่ยนแปลงการควบคุม |
| **ข้อกำหนดการชำระเงิน** | เงื่อนไขการชำระเงิน ดอกเบี้ยกรณีชำระล่าช้า ภาษี การปรับราคา |

## กรอบงานการจัดลำดับความสำคัญในการเจรจา

**ระดับ 1 — สิ่งที่ต้องได้ (จุดที่ทำให้ข้อตกลงล้มเหลว)**
ประเด็นที่ไม่สามารถดำเนินการต่อได้หากไม่ได้รับการแก้ไข เช่น ความรับผิดที่ไม่มีวงเงินจำกัด ข้อกำหนดทรัพย์สินทางปัญญาที่มีความเสี่ยง

**ระดับ 2 — สิ่งที่ควรได้ (ความต้องการที่แข็งแกร่ง)**
ประเด็นที่ส่งผลกระทบต่อความเสี่ยงแต่มีช่องว่างในการเจรจา เช่น การปรับวงเงินจำกัดความรับผิด

**ระดับ 3 — สิ่งที่ดีถ้าได้ (จุดที่สามารถยอมได้)**
ประเด็นที่ปรับปรุงสถานะแต่สามารถยอมรับได้เชิงกลยุทธ์

**กลยุทธ์:** เจรจาประเด็นระดับ 1 ก่อน แลกเปลี่ยนประเด็นระดับ 3 เพื่อให้ได้ประเด็นระดับ 2 ห้ามยอมแพ้ประเด็นระดับ 1 โดยไม่แจ้งเตือนระดับสูง

```
การวิเคราะห์ทีละข้อ (สีเขียว/สีเหลือง/สีแดง)
  → สำหรับข้อความสีเหลืองและสีแดง: ข้อเสนอแก้ไขที่เจาะจง + ทางเลือกสำรอง + เหตุผล
กลยุทธ์การเจรจา (แนวทาง ลำดับความสำคัญ ประเด็นที่สามารถยอมได้)

```
