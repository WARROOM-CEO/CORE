# การสรุปข้อมูลสำหรับทีมกฎหมาย — การสรุปข้อมูลสำหรับทีมกฎหมาย

**สร้างเอกสารสรุปเชิงบริบทสำหรับงานด้านกฎหมาย — สรุปรายวัน การวิจัยหัวข้อ หรือการตอบสนองเหตุการณ์ฉุกเฉิน**

---

## สกิลนี้ทำอะไร?

สร้างเอกสารสรุปที่เหมาะสมกับบริบทสำหรับงานด้านกฎหมายใน 3 โหมด: สรุปรายวันเพื่อเริ่มต้นวันทำงาน เอกสารสรุปหัวข้อสำหรับการวิจัยคำถามกฎหมายเฉพาะเจาะจง และเอกสารสรุปเหตุการณ์สำหรับสถานการณ์เร่งด่วน

> **หมายเหตุสำคัญ**: คำสั่งนี้ช่วยในการทำงานด้านกฎหมายแต่ไม่ใช่การให้คำแนะนำทางกฎหมาย เอกสารสรุปทั้งหมดควรได้รับการตรวจสอบโดยผู้เชี่ยวชาญด้านกฎหมายที่มีคุณวุฒิก่อนนำไปใช้

---

## วิธีใช้

```
/brief daily              # สรุปรายวันเกี่ยวกับเรื่องกฎหมายยามเช้า
/brief topic [คำถาม]      # วิจัยเอกสารสรุปสำหรับคำถามกฎหมายเฉพาะเจาะจง
/brief incident [หัวข้อ]  # เอกสารสรุปเร่งด่วนสำหรับสถานการณ์ที่กำลังพัฒนา
```

---

## 3 โหมด

### 📋 สรุปรายวัน
สรุปยามเช้าของรายการที่เกี่ยวข้องกับกฎหมายทั้งหมดที่ต้องรู้เพื่อเริ่มวันทำงาน

**แหล่งที่ตรวจสอบ:**
- **อีเมล** — คำขอตรวจสอบสัญญา คำถามเกี่ยวกับการปฏิบัติตามกฎระเบียบ การตอบกลับจากคู่สัญญา สื่อสารจากทนายความภายนอก
- **ปฏิทิน** — การประชุมที่ต้องการการเตรียมการด้านกฎหมายวันนี้ กำหนดเวลาภายในสัปดาห์นี้
- **แชท** — ข้อความค้างคืนในช่องทางของทีมกฎหมาย การขอความเห็นด้านกฎหมาย
- **ระบบจัดการสัญญา** — สัญญาที่รอการตรวจสอบหรือลายเซ็น วันหมดอายุที่ใกล้ถึง
- **ระบบจัดการลูกค้าสัมพันธ์** — ข้อตกลงที่เคลื่อนเข้าสู่ขั้นตอนที่ต้องการการมีส่วนร่วมด้านกฎหมาย

**ผลลัพธ์ที่ได้:**
- เร่งด่วน/ต้องดำเนินการทันที
- กระบวนการสัญญา (รอการตรวจสอบ รอคู่สัญญา ใกล้ถึงกำหนดเวลา)
- คำขอใหม่ที่เข้ามา
- กำหนดการวันนี้พร้อมการเตรียมการที่ต้องการ
- กำหนดเวลาภายในสัปดาห์นี้

---

### 🔍 เอกสารสรุปหัวข้อ
วิจัยและสร้างเอกสารสรุปเกี่ยวกับคำถามหรือหัวข้อกฎหมายเฉพาะเจาะจงจากแหล่งข้อมูลที่มีอยู่

**ขั้นตอน:** รับหัวข้อ → ค้นหาทั่วทุกแหล่ง (เอกสาร อีเมล แชท ระบบจัดการสัญญา) → สังเคราะห์เป็นเอกสารสรุปที่มีโครงสร้าง

**ผลลัพธ์:** สรุป → พื้นหลัง → สถานะปัจจุบัน → ประเด็นสำคัญที่ต้องพิจารณา → ตัวอย่างก่อนหน้าภายในองค์กร → ช่องว่าง → ขั้นตอนต่อไปที่แนะนำ

**หมายเหตุ:** เอกสารสรุปหัวข้อสังเคราะห์จากแหล่งข้อมูลที่มีอยู่เท่านั้น ไม่ใช่การวิจัยกฎหมายเชิงทางการ ถ้าหัวข้อต้องการคำพิพากษาศาลหรือแหล่งอ้างอิงกฎหมายปัจจุบัน ควรใช้ Westlaw, Lexis หรือปรึกษาทนายความภายนอก

---

### 🚨 เอกสารสรุปเหตุการณ์
เอกสารสรุปเร่งด่วนสำหรับสถานการณ์ที่ต้องการความสนใจด้านกฎหมายทันที เช่น การละเมิดข้อมูล ภัยคุกคามการฟ้องร้อง การสอบสวนของหน่วยงานกำกับดูแล

**ขั้นตอน:** รับหัวข้อเหตุการณ์ → ตรวจสอบแหล่งข้อมูลทั้งหมดอย่างรวดเร็ว (อีเมล แชท เอกสาร ระบบจัดการสัญญา) → สร้างเอกสารสรุปเหตุการณ์พร้อมกิจกรรมทันที

**ผลลัพธ์:** สรุปสถานการณ์ → ลำดับเวลา → ประเด็นกฎหมายที่ต้องพิจารณาทันที → ข้อตกลงที่เกี่ยวข้อง → ผู้ติดต่อสำคัญ → กิจกรรมทันทีที่แนะนำ → ช่องว่างข้อมูล

**ข้อควรระวังสำหรับเอกสารสรุปเหตุการณ์:**
- ความเร็วสำคัญ — ผลิตเอกสารสรุปด้วยข้อมูลที่มีอยู่ ไม่รอให้ข้อมูลครบถ้วน
- แจ้งการระงับการทำลายหลักฐานหรือภาระผูกพันในการเก็บรักษาหลักฐานทันที
- ระบุประเด็นเกี่ยวกับสิทธิประโยชน์ (สิทธิประโยชน์ระหว่างทนายความกับลูกความ / งานที่ได้รับการคุ้มครอง)
- ถ้าเป็นการละเมิดข้อมูล → แจ้งกำหนดเวลาแจ้งเตือนภายใต้ GDPR 72 ชั่วโมงทันที

---

## หมายเหตุทั่วไป

- ถ้าแหล่งข้อมูลไม่พร้อม — ระบุช่องว่างชัดเจนว่าตรวจสอบอะไรไม่ได้
- เอกสารสรุปทุกฉบับควรสามารถดำเนินการได้: ทุกรายการต้องมีขั้นตอนต่อไปหรือเหตุผลที่รวมไว้
- สรุปรายวัน — เรียนรู้ความชอบของคุณเมื่อเวลาผ่านไป (สิ่งที่พบว่ามีประโยชน์ สิ่งที่ต้องการกรองออก)
