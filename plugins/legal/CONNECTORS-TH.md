# คู่มือการเชื่อมต่อเครื่องมือ (ภาษาไทย)

## วิธีที่ปลั๊กอินอ้างอิงเครื่องมือ

ไฟล์ในปลั๊กอินใช้ `~~หมวดหมู่` เป็นตัวแทนชั่วคราวสำหรับเครื่องมือที่ผู้ใช้เชื่อมต่อในหมวดนั้น ตัวอย่างเช่น `~~ระบบจัดเก็บบนคลาวด์` อาจหมายถึง Box, Egnyte หรือผู้ให้บริการจัดเก็บข้อมูลอื่นที่มี MCP server

ปลั๊กอินออกแบบมาให้ **ไม่ผูกกับเครื่องมือเฉพาะเจาะจง** — อธิบายกระบวนการทำงานในรูปแบบหมวดหมู่ (ระบบจัดเก็บบนคลาวด์, ระบบแชท, ชุดโปรแกรมสำนักงาน ฯลฯ) แทนที่จะระบุผลิตภัณฑ์เฉพาะเจาะจง ไฟล์ `.mcp.json` จะกำหนด MCP server เริ่มต้นไว้ล่วงหน้า แต่ MCP server ใดก็ตามในหมวดหมู่นั้นสามารถใช้งานได้
