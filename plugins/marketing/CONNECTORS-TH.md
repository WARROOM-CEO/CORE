## วิธีการทำงานของการอ้างอิงเครื่องมือ

ปลั๊กอินใช้ `~~หมวดหมู่` เป็นตัวแทนชั่วคราวสำหรับเครื่องมือที่ผู้ใช้เชื่อมต่อในแต่ละหมวด เช่น `~~ระบบการตลาดอัตโนมัติ` อาจหมายถึง HubSpot, Marketo หรือแพลตฟอร์มการตลาดอื่นๆ ที่มี MCP server

## ประโยชน์ของการใช้แต่ละเครื่องมือ

| เครื่องมือ | ใช้สำหรับ |
|-----------|---------|
| **Slack** | แชร์ร่างเอกสาร รายงาน และเอกสารสรุปกับทีม |
| **Canva / Figma** | สร้างและแก้ไขสื่อภาพ จัดการไฟล์แบรนด์ |
| **HubSpot** | ดึงข้อมูลแคมเปญ จัดการรายชื่อผู้ติดต่อ ติดตามระบบการตลาดอัตโนมัติ |
| **Amplitude** | ดึงข้อมูลการวิเคราะห์ผลิตภัณฑ์และพฤติกรรมผู้ใช้สำหรับรายงานประสิทธิภาพ |
| **Notion** | เข้าถึงเอกสารสรุป คู่มือสไตล์ และเอกสารแคมเปญ |
| **Ahrefs / Similarweb** | วิจัยคำหลัก SEO วิเคราะห์ลิงก์ย้อนกลับ ตรวจสอบเว็บไซต์ วิเคราะห์คู่แข่ง |
| **Klaviyo** | ร่างและตรวจสอบลำดับอีเมลและแคมเปญ |
| **Supermetrics** | ดึงข้อมูลการตลาดจากหลายแพลตฟอร์มสำหรับการวิเคราะห์ |
