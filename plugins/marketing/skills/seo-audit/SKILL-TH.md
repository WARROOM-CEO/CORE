# การตรวจสอบ SEO — ตรวจสอบ SEO ครบวงจร

**ดำเนินการตรวจสอบ SEO ครบถ้วน — วิจัยคำหลัก การวิเคราะห์บนหน้า เปรียบเทียบช่องว่างเนื้อหา การตรวจสอบทางเทคนิค และการเปรียบเทียบกับคู่แข่ง**

---

## สกิลนี้ทำอะไร?

ตรวจสอบสุขภาพ SEO ของเว็บไซต์ วิจัยโอกาสคำหลัก ระบุช่องว่างเนื้อหา และเปรียบเทียบกับคู่แข่ง พร้อมแผนปฏิบัติการที่จัดลำดับตามชัยชนะรวดเร็วและการลงทุนระยะยาว

---

## วิธีใช้

```
/seo-audit
/seo-audit <URL หรือหัวข้อ> [ประเภทการตรวจสอบ]
```

## ประเภทการตรวจสอบที่รองรับ

| ประเภท | ขอบเขต |
|--------|--------|
| **การตรวจสอบเว็บไซต์ทั้งหมด** | การทบทวน SEO ครบถ้วนทุกส่วน |
| **การวิจัยคำหลัก** | ค้นหาโอกาสคำหลักสำหรับหัวข้อหรือโดเมน |
| **การวิเคราะห์ช่องว่างเนื้อหา** | หาหัวข้อที่คู่แข่งจัดอันดับได้แต่คุณยังไม่มี |
| **การตรวจสอบ SEO ทางเทคนิค** | การเข้าถึงได้ ความเร็ว ข้อมูลโครงสร้าง โครงสร้างพื้นฐาน |
| **การเปรียบเทียบ SEO กับคู่แข่ง** | เปรียบเทียบ SEO แบบเจาะจงกับคู่แข่งที่ระบุ |

ถ้าไม่ระบุ: ค่าเริ่มต้นคือ **การตรวจสอบเว็บไซต์ทั้งหมด**

---

## สิ่งที่วิเคราะห์ในการตรวจสอบแบบครบถ้วน

### 1. การวิจัยคำหลัก
- **คำหลักหลัก** — มีเจตนาสูง เกี่ยวข้องกับผลิตภัณฑ์/บริการโดยตรง
- **คำหลักรอง** — คำศัพท์สนับสนุน
- **โอกาสคำหลักยาว** — เฉพาะเจาะจง การแข่งขันต่ำ
- **คำหลักคำถาม** — "วิธี...", "คืออะไร..." (ผู้คนถามเพิ่มเติม)
- **การจัดหมวดหมู่เจตนา**: ให้ข้อมูล นำทาง เชิงพาณิชย์ การทำธุรกรรม

### 2. SEO บนหน้า
- **Title tag**: ≤60 ตัวอักษร มีคำหลัก
- **Meta description**: ≤160 ตัวอักษร มีคำกระตุ้นการดำเนินการ
- **โครงสร้าง H1/H2/H3**
- **การใช้คำหลัก** — ใน 100 คำแรก ในหัวข้อย่อย
- **การเชื่อมโยงภายใน/ภายนอก**
- **ข้อความแทนรูปภาพ**
- **โครงสร้าง URL**

### 3. การวิเคราะห์ช่องว่างเนื้อหา
- หัวข้อที่คู่แข่งจัดอันดับได้แต่คุณยังไม่มี
- เนื้อหาที่ไม่ได้อัปเดตใน 12+ เดือน
- เนื้อหาบาง (ข้อมูลน้อยเกินไป)
- รูปแบบที่คู่แข่งใช้แต่คุณยังไม่มี
- โอกาสคลัสเตอร์หัวข้อ

### 4. รายการตรวจสอบ SEO ทางเทคนิค
- ความเร็วหน้า ความเป็นมิตรกับมือถือ
- ข้อมูลโครงสร้าง (FAQ, HowTo, schema บทความ)
- การเข้าถึงได้ — robots.txt sitemap แท็ก canonical
- ลิงก์เสีย HTTPS Core Web Vitals
- ปัญหาการทำดัชนี

### 5. การเปรียบเทียบ SEO กับคู่แข่ง
- การทับซ้อนคำหลักและช่องว่าง
- สัญญาณอำนาจโดเมน
- ความลึกเนื้อหาและความถี่ในการเผยแพร่
- โปรไฟล์ลิงก์ย้อนกลับ
- การครอบครองคุณสมบัติในหน้าผลการค้นหา

## ผลลัพธ์ที่จะได้รับ

| ส่วน | รายละเอียด |
|---------|-----------|
| **สรุปผู้บริหาร** | 3-5 ประโยค จุดแข็งสำคัญ 3 อันดับแรก |
| **ตารางโอกาสคำหลัก** | 15-25 คำหลัก พร้อมระดับความยาก คะแนนโอกาส เจตนา |
| **ตารางปัญหา SEO บนหน้า** | ปัญหาแต่ละหน้า + ระดับความรุนแรง + คำแนะนำแก้ไข |
| **คำแนะนำช่องว่างเนื้อหา** | หัวข้อ เหตุผล รูปแบบแนะนำ ลำดับความสำคัญ ความพยายาม |
| **รายการตรวจสอบ SEO ทางเทคนิค** | ผ่าน/ไม่ผ่าน/คำเตือน ทุกข้อตรวจสอบ |
| **สรุปการเปรียบเทียบคู่แข่ง** | เปรียบเทียบมิติสำคัญ |
| **แผนปฏิบัติการที่จัดลำดับแล้ว** | ชัยชนะรวดเร็ว (ทำสัปดาห์นี้) + การลงทุนเชิงกลยุทธ์ (ไตรมาสนี้) |

---

## หลังจากการตรวจสอบแล้ว Claude จะสอบถาม

- ต้องการร่างเอกสารคำชี้แจงเนื้อหาสำหรับโอกาสคำหลักอันดับต้นไหม?
- ต้องการปรับปรุง title tag และ meta description สำหรับหน้าสำคัญไหม?
- ต้องการปฏิทินเนื้อหาจากผลการวิเคราะห์ช่องว่างไหม?
- ต้องการทำการวิเคราะห์เดียวกันกับคู่แข่งหรือโดเมนอื่นไหม?
