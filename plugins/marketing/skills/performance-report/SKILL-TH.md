# รายงานผลการทำงาน — รายงานผลการทำงานการตลาด

**สร้างรายงานผลการทำงานการตลาดพร้อมตัวชี้วัดหลัก การวิเคราะห์แนวโน้ม สิ่งที่ได้ผลและสิ่งที่ควรปรับปรุง พร้อมคำแนะนำการปรับปรุงที่จัดลำดับความสำคัญแล้ว**

---

## สกิลนี้ทำอะไร?

สร้างรายงานผลการทำงานการตลาดที่มีโครงสร้าง พร้อมตัวชี้วัดหลัก การวิเคราะห์แนวโน้ม ข้อมูลเชิงลึก และคำแนะนำการปรับปรุงสำหรับทุกประเภทรายงาน

## ตัวชี้วัดหลักตามช่องทาง

### อีเมล
อัตราการเปิด อัตราการคลิก อัตราการคลิกจากผู้เปิด อัตราการยกเลิกการสมัคร อัตราการแปลงยอดขาย อัตราการเติบโตของรายชื่อ

### โซเชียลมีเดีย
จำนวนการแสดงผล อัตราการมีส่วนร่วม การเติบโตของผู้ติดตาม อัตราการแชร์ อัตราการดูวิดีโอจนจบ

### โฆษณาแบบเสียเงิน (ค้นหาและโซเชียล)
อัตราการคลิก ต้นทุนต่อการคลิก ต้นทุนต่อหนึ่งพันการแสดงผล ต้นทุนต่อการได้มา ผลตอบแทนจากการใช้จ่าย ความถี่ คะแนนคุณภาพ

### SEO / การเข้าชมแบบธรรมชาติ
จำนวนเซสชันแบบออร์แกนิก อันดับคำหลัก อำนาจโดเมน ลิงก์ย้อนกลับ ความเร็วในการโหลดหน้า

### การตลาดโดยรวม
ลีดคุณภาพระดับการตลาด ลีดคุณภาพระดับการขาย ยอดขายที่สร้างได้ ต้นทุนการได้ลูกค้าใหม่ รายได้ที่มาจากการตลาด

---

## โครงสร้างรายงานที่จะได้รับ

1. **สรุปผู้บริหาร** — 2-3 ประโยค ตัวชี้วัดหลัก สิ่งที่ได้ผล/ข้อกังวลหลัก
2. **แผงควบคุมตัวชี้วัดหลัก** — ตารางเปรียบเทียบกับช่วงก่อนหน้า + เป้าหมาย
3. **การวิเคราะห์แนวโน้ม** — จุดเปลี่ยน รูปแบบตามฤดูกาล การเปรียบเทียบกับเกณฑ์มาตรฐาน
4. **สิ่งที่ได้ผล** — 3-5 สิ่งที่ประสบความสำเร็จ พร้อมข้อมูลและสมมติฐาน
5. **สิ่งที่ควรปรับปรุง** — 3-5 สิ่งที่ไม่ได้ผล พร้อมคำแนะนำ
6. **ข้อมูลเชิงลึกและข้อสังเกต** — รูปแบบที่มองไม่เห็นจากตัวชี้วัดเพียงอย่างเดียว
7. **คำแนะนำ** — จัดลำดับตามแมทริกซ์ผลกระทบเทียบกับความพยายาม
8. **จุดเน้นช่วงถัดไป** — 3 ลำดับความสำคัญหลัก + การทดลองที่ควรทำ

## แมทริกซ์ผลกระทบเทียบกับความพยายาม

| | ความพยายามต่ำ | ความพยายามสูง |
|---|---|---|
| **ผลกระทบสูง** | ดำเนินการก่อน | วางแผนสำหรับรอบถัดไป |
| **ผลกระทบต่ำ** | ทำเมื่อมีเวลา | ลดลำดับความสำคัญ |

---

## ความถี่ของรายงาน

| ความถี่ | เนื้อหาหลัก |
|---------|-----------|
| **รายสัปดาห์** | 3 ตัวชี้วัดหลัก + สิ่งที่ได้ผล + ลำดับความสำคัญสัปดาห์นี้ |
| **รายเดือน** | สรุปผู้บริหาร + สรุปช่องทาง + ไฮไลท์แคมเปญ + คำแนะนำ |
| **รายไตรมาส (QBR)** | ไตรมาสเทียบกับเป้าหมาย + แนวโน้มปีนี้ + ROI ช่องทาง + คำแนะนำเชิงกลยุทธ์ |
