## สกิลนี้ทำอะไร?

เป็นคลังความรู้และกรอบการทำงานสำหรับสร้างเนื้อหาการตลาดที่มีประสิทธิภาพในทุกช่องทาง ครอบคลุมเทมเพลต แนวทางปฏิบัติที่ดีที่สุด การทำ SEO สูตรหัวข้อ และคำกระตุ้นการดำเนินการ

---

## ประเภทเนื้อหาที่รองรับ

| ประเภท | โครงสร้าง |
|--------|---------|
| **บทความบล็อก** | หัวข้อ → บทนำ (ดึงดูด) → 3-5 ส่วน → สรุป → คำอธิบายเมตา |
| **โพสต์โซเชียลมีเดีย** | ประโยคดึงดูด → เนื้อหาหลัก → คำกระตุ้นการดำเนินการ → แฮชแท็ก |
| **จดหมายข่าวอีเมล** | หัวเรื่อง → ข้อความแสดงตัวอย่าง → ส่วนหัว → ส่วนเนื้อหา → คำกระตุ้นการดำเนินการ → ส่วนท้าย |
| **หน้า Landing Page** | หัวข้อ → หัวข้อย่อย → ส่วนฮีโร → ข้อเสนอคุณค่า → หลักฐานความน่าเชื่อถือ → คำถามที่พบบ่อย → คำกระตุ้นการดำเนินการสุดท้าย |
| **ข่าวประชาสัมพันธ์** | หัวข้อ → หัวข้อย่อย → วันที่ → ย่อหน้าแรก → เนื้อหาหลัก → ข้อมูลบริษัท → ข้อมูลติดต่อ |
| **กรณีศึกษา** | ชื่อเรื่อง (ผลลัพธ์) → สรุปภาพรวม → ความท้าทาย → วิธีแก้ไข → ผลลัพธ์ → คำพูดอ้างอิง → คำกระตุ้นการดำเนินการ |

## แนวทางปฏิบัติที่ดีที่สุดตามช่องทาง

### บทความบล็อก
- เขียนในระดับการอ่านชั้นมัธยมศึกษาปีที่ 3-6 สำหรับกลุ่มทั่วไป
- ย่อหน้าสั้น (2-4 ประโยค) หัวข้อย่อยทุก 200-300 คำ
- ใช้รายการหัวข้อย่อยและรายการลำดับเลขเพื่อแบ่งข้อมูล
- เขียนในรูปแบบประโยคที่มีผู้กระทำ

### โซเชียลมีเดีย
- **LinkedIn**: เป็นมืออาชีพแต่เป็นกันเอง ย่อหน้าสั้น เรื่องราวส่วนตัวได้ผลดี
- **Twitter/X**: กระชับ ประโยคเปิดต้องแข็งแกร่ง
- **Instagram**: เน้นภาพเป็นหลัก การเล่าเรื่อง แฮชแท็กอยู่ท้าย

### อีเมล
- หัวเรื่องสร้างความอยากรู้หรือระบุคุณค่าชัดเจน
- คำกระตุ้นการดำเนินการหลักเพียงหนึ่งรายการต่ออีเมล
- เน้นบนมือถือ: เนื้อหาส่วนใหญ่อ่านบนโทรศัพท์มือถือ

---

## พื้นฐานการทำ SEO

### การใช้คำหลัก
- คำหลักหลัก 1 คำ + คำหลักรอง 2-3 คำต่อชิ้นงาน
- ใช้คำหลักหลักใน: หัวข้อ ย่อหน้าแรก หัวข้อย่อยหนึ่งรายการ คำอธิบายเมตา
- เขียนเพื่อมนุษย์ก่อน — อย่ายัดคำหลัก

### รายการตรวจสอบ SEO บนหน้า
- Title tag: ≤60 ตัวอักษร มีคำหลักหลัก
- Meta description: ≤160 ตัวอักษร มีคำหลัก ดึงดูดให้คลิก
- H1: หนึ่งรายการต่อหน้า
- ข้อความแทนรูปภาพ: มีคำอธิบาย

## สูตรหัวข้อที่ได้ผลดี

- **"วิธี [บรรลุผล] โดยไม่ต้อง [อุปสรรคทั่วไป]"**
- **"[จำนวน] วิธีที่พิสูจน์แล้วในการ [บรรลุผล]"**
- **"ทำไม [ความเชื่อทั่วไป] ถึงผิด (และควรทำอะไรแทน)"**
- **"คู่มือ[ครบถ้วน]สำหรับ [หัวข้อ]"**

---

## แนวทางปฏิบัติที่ดีที่สุดสำหรับคำกระตุ้นการดำเนินการ

- ใช้คำกริยาแสดงการกระทำ: "รับ", "เริ่ม", "ดาวน์โหลด", "ลอง", "ดู"
- ระบุชัดเจนว่าขั้นตอนต่อไปคืออะไร
- ลดอุปสรรค: "ไม่ต้องใช้บัตรเครดิต", "ยกเลิกได้ตลอดเวลา"
- คำกระตุ้นการดำเนินการหลักเพียงหนึ่งรายการต่อหน้า/อีเมล
