# การร่างเนื้อหา — ร่างเนื้อหาการตลาด

**ร่างบทความบล็อก โพสต์โซเชียลมีเดีย จดหมายข่าวอีเมล หน้า Landing Page ข่าวประชาสัมพันธ์ และกรณีศึกษา พร้อมรูปแบบเฉพาะช่องทางและคำแนะนำการทำ SEO**

## สกิลนี้ทำอะไร?

## ข้อมูลที่ Claude จะสอบถาม

| ข้อมูล | รายละเอียด |
|--------|-----------|
| **ประเภทเนื้อหา** | บทความบล็อก โซเชียลมีเดีย (ระบุแพลตฟอร์ม) จดหมายข่าวอีเมล หน้า Landing Page ข่าวประชาสัมพันธ์ กรณีศึกษา |
| **หัวข้อ** | เรื่องหรือแนวคิดหลักของเนื้อหา |
| **กลุ่มเป้าหมาย** | ใครจะอ่าน — ตำแหน่งงาน อุตสาหกรรม จุดปวด |
| **ข้อความหลัก** | 2-4 ประเด็นสำคัญที่ต้องการสื่อสาร |
| **น้ำเสียง** | เช่น เป็นทางการ สื่อสารแบบกันเอง ทางเทคนิค (ไม่จำเป็นถ้ามีน้ำเสียงแบรนด์กำหนดไว้) |
| **ความยาว** | เป้าหมายจำนวนคำหรือข้อจำกัด เช่น "1000 คำ", "280 ตัวอักษร" |

## สิ่งที่ได้รับตามประเภทเนื้อหา

### บทความบล็อก
- หัวข้อ (เสนอ 2-3 ตัวเลือก)
- บทนำพร้อมประโยคดึงดูด
- 3-5 ส่วนพร้อมหัวข้อย่อย
- สรุปพร้อมคำกระตุ้นการดำเนินการ
- คำแนะนำการทำ SEO + คำหลักหลัก

### โพสต์โซเชียลมีเดีย
- รูปแบบที่เหมาะสมกับแพลตฟอร์ม
- ประโยคดึงดูดในบรรทัดแรก
- แฮชแท็ก 3-5 รายการที่เกี่ยวข้อง
- คำกระตุ้นการดำเนินการหรือคำกระตุ้นให้มีส่วนร่วม

### จดหมายข่าวอีเมล
- หัวเรื่อง (2-3 ตัวเลือก)
- ข้อความแสดงตัวอย่าง
- ส่วนเนื้อหา
- ข้อความปุ่มสำหรับคำกระตุ้นการดำเนินการ

### หน้า Landing Page
- หัวข้อ + หัวข้อย่อย
- เนื้อหาส่วนฮีโร่
- ข้อเสนอคุณค่า 3-4 ข้อ
- คำกระตุ้นการดำเนินการหลักและรอง
- Title tag และ meta description

### ข่าวประชาสัมพันธ์
- หัวข้อตามมาตรฐานข่าว
- วันที่ + ย่อหน้าแรก
- ที่ว่างสำหรับคำพูดอ้างอิงและข้อมูลบริษัท

### กรณีศึกษา
- ชื่อเรื่องที่เน้นผลลัพธ์
- ความท้าทาย → วิธีแก้ไข → ผลลัพธ์
- ที่ว่างสำหรับคำพูดอ้างอิงจากลูกค้า
