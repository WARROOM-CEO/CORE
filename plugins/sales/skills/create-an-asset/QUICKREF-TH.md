# การสร้างสื่อการขาย — คู่มือย่อ (ภาษาไทย)

## วิธีเรียกใช้
```
/create-an-asset
/create-an-asset [ชื่อบริษัท]
"สร้างสื่อการขายสำหรับ [บริษัท]"
```

---

## ข้อมูลที่ต้องการ

| ข้อมูล | สิ่งที่ต้องระบุ |
|-------|----------------|
| **(ก) ลูกค้าเป้าหมาย** | บริษัท ผู้ติดต่อ ขั้นตอนดีล จุดปวด คำถอดเสียงการประชุม |
| **(ข) กลุ่มผู้ชม** | ผู้บริหาร / ทีมเทคนิค / ทีมปฏิบัติการ / ผสม + สิ่งที่พวกเขาสนใจ |
| **(ค) วัตถุประสงค์** | แนะนำ / ติดตามผล / เจาะลึก / ปรับแนวร่วม / POC / ปิดดีล |
| **(ง) รูปแบบ** | หน้า Landing Page / พรีเซนเทชัน / แผ่นพับข้อมูล / การสาธิตกระบวนการทำงาน |

---

## เลือกรูปแบบ

| ถ้าต้องการ... | เลือก... |
|--------------|---------|
| ประสบการณ์หลายแท็บที่น่าประทับใจ | **หน้า Landing Page แบบ Interactive** |
| สื่อสำหรับนำเสนอในการประชุม | **พรีเซนเทชัน** |
| สรุปสั้นๆ ที่ฝากไว้ให้ลูกค้า | **แผ่นพับข้อมูล (One-Pager)** |
| แสดงให้เห็นว่าระบบเชื่อมต่อกันอย่างไร | **การสาธิตกระบวนการทำงาน** |

---

## ตัวอย่างคำสั่ง

**แบบง่าย:**
```
สร้างสื่อการขายสำหรับบริษัท Acme Corp
```

**พร้อมบริบท:**
```
สร้างสื่อการขายสำหรับบริษัท Acme Corp พวกเขาเป็นบริษัทผลิต
ที่มีปัญหาเรื่องห่วงโซ่อุปทาน ประชุมกับ COO สัปดาห์ที่แล้ว
ต้องการสื่อบางอย่างสำหรับทีมผู้บริหาร
```

**การสาธิตกระบวนการทำงาน:**
```
ทำแผนภาพกระบวนการสำหรับบริษัท TechStart แสดงว่าพวกเขาจะใช้ผลิตภัณฑ์ของเรา
ในการตรวจสอบการปฏิบัติตามสัญญาอย่างไร
```

## หลังจากสร้างแล้ว

| ต้องการ... | พูดว่า... |
|-----------|---------|
| เปลี่ยนสี | "ใช้สีแบรนด์ของเราแทน" |
| เพิ่มส่วน | "เพิ่มส่วนเรื่องความปลอดภัย" |
| ย่อให้สั้น | "ทำให้กระชับขึ้น" |
| แก้ไขข้อมูล | "ชื่อ CEO ผิด ชื่อจริงคือ Jane Smith" |
| ได้ไฟล์ PDF | "ทำเวอร์ชันสำหรับพิมพ์" |
