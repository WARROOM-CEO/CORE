# การสร้างสื่อการขาย

**สำหรับทีมขายทุกที่**

สร้างสื่อการขายแบบมืออาชีพที่พร้อมแชร์กับลูกค้าในไม่กี่นาที ไม่ต้องมีทักษะด้านการออกแบบ

---

## สกิลนี้ทำอะไร?

สร้างสื่อการขายที่ปรับแต่งตามลูกค้าเป้าหมาย โดยสอบถามข้อมูลจากคุณเกี่ยวกับ:
1. **ลูกค้าเป้าหมาย** — พวกเขาคือใคร คุณได้พูดคุยอะไรกันไปแล้วบ้าง
2. **กลุ่มผู้ชม** — ใครจะเป็นผู้ดู พวกเขาสนใจเรื่องอะไร
3. **วัตถุประสงค์** — คุณต้องการบรรลุเป้าหมายอะไร
4. **รูปแบบ** — คุณต้องการนำเสนออย่างไร

จากนั้นทำการวิจัย เขียนเนื้อหา ออกแบบ และสร้างสื่อที่พร้อมแชร์กับลูกค้า

---

## รูปแบบที่รองรับ

| รูปแบบ | เหมาะสำหรับ | ผลลัพธ์ |
|--------|------------|--------|
| **หน้า Landing Page แบบ Interactive** | การประชุมผู้บริหาร การนำเสนอข้อเสนอคุณค่า | หน้าหลายแท็บพร้อมการสาธิตและเครื่องคำนวณ |
| **พรีเซนเทชัน** | การนำเสนอทางการ กลุ่มผู้ฟังขนาดใหญ่ | สไลด์แนวตั้งพร้อมระบบนำทาง |
| **แผ่นพับข้อมูล (One-Pager)** | เอกสารฝากไว้ สรุปอย่างรวดเร็ว | สรุปผู้บริหารที่เลื่อนได้ในหน้าเดียว |
| **การสาธิตกระบวนการทำงาน / สถาปัตยกรรม** | การเจาะลึกทางเทคนิค ข้อเสนอ POC | แผนภาพแบบ interactive พร้อมการเคลื่อนไหวแสดงลำดับงาน |

## วิธีเริ่มต้น

### ตัวเลือกที่ 1: คำสั่งง่าย
```
สร้างสื่อการขายสำหรับบริษัท Acme Corp
```

### ตัวเลือกที่ 2: พร้อมบริบท
```
สร้างสื่อการขายสำหรับบริษัท Acme Corp ฉันประชุมกับผู้จัดการฝ่ายวิศวกรรม
สัปดาห์ที่แล้ว — พวกเขามีปัญหาเรื่องวงรอบการเปิดตัวผลิตภัณฑ์ช้า
ต้องการสื่อบางอย่างสำหรับการติดตามผลกับทีมเทคนิค
```

### ตัวเลือกที่ 3: การสาธิตกระบวนการทำงาน
```
ทำแผนภาพกระบวนการแสดงว่าลูกค้าจะใช้ผลิตภัณฑ์ของเราในการ
ประมวลผลใบแจ้งหนี้อัตโนมัติอย่างไร: ใบแจ้งหนี้เข้าทางอีเมล
→ AI ดึงข้อมูล → ตรวจสอบกับระบบ ERP → แจ้งเตือนกรณีผิดปกติ
```

## รายละเอียดผลลัพธ์แต่ละรูปแบบ

### หน้า Landing Page แบบ Interactive
- ระบบนำทางแบบแท็บ
- ข้อมูลบริษัทและการวิจัย
- การสาธิตกรณีใช้งาน
- เครื่องคำนวณ ROI
- ธีมมืดแบบมืออาชีพพร้อมสีแบรนด์ของลูกค้าเป้าหมาย

### พรีเซนเทชัน
- สไลด์ชื่อเรื่องพร้อมโลโก้ทั้งสองฝ่าย
- วาระการประชุม
- สไลด์เนื้อหา (หนึ่งประเด็นต่อสไลด์)
- สรุปและขั้นตอนต่อไป
- โน้ตสำหรับผู้นำเสนอครบถ้วน

### แผ่นพับข้อมูล (One-Pager)
- คำแถลงหลัก
- 3 จุดคุณค่าสำคัญ
- หลักฐานสนับสนุน
- คำกระตุ้นการดำเนินการชัดเจน

### การสาธิตกระบวนการทำงาน
- โหนดส่วนประกอบแบบภาพ
- การไหลของข้อมูลแบบเคลื่อนไหว
- คำอธิบายทีละขั้นตอน
- ปุ่มควบคุม เล่น/หยุดชั่วคราว/ทีละขั้น
- ข้อมูลตัวอย่างในแต่ละขั้นตอน

## กระบวนการทำงาน

```
1. คุณให้บริบท (ลูกค้าเป้าหมาย กลุ่มผู้ชม วัตถุประสงค์)
        ↓
2. Claude วิจัยลูกค้าเป้าหมาย
        ↓
3. Claude ถาม 3-4 คำถามเพื่อยืนยันแนวทาง
        ↓
4. คุณยืนยัน
        ↓
5. Claude สร้างสื่อการขาย
        ↓
6. คุณปรับแต่งตามต้องการ
```

## การแชร์สื่อการขาย

ผลลัพธ์เป็นไฟล์ HTML แบบครบวงจร สามารถแชร์ได้โดย:

- **โฮสต์แบบคงที่**: อัปโหลดไปยัง Netlify, Vercel, GitHub Pages
- **ป้องกันด้วยรหัสผ่าน**: ผู้ให้บริการโฮสต์ส่วนใหญ่มีระบบป้องกันด้วยรหัสผ่าน
- **ส่งตรง**: ส่งไฟล์ HTML ทางอีเมล — ใช้งานได้แบบออฟไลน์
- **ฝังในเว็บ**: ใช้ iframe ในหน้าเว็บอื่น

## เคล็ดลับสำหรับผลลัพธ์ที่ดีที่สุด

**ให้บริบทที่ครบถ้วน** — ยิ่งให้ข้อมูลมากเกี่ยวกับบทสนทนาที่ผ่านมา จุดปวด และความกังวลของผู้มีส่วนได้ส่วนเสีย สื่อการขายก็จะยิ่งปรับแต่งเฉพาะบุคคลมากขึ้น

**อัปโหลดคำถอดเสียง** — ถ้ามีบันทึกการประชุม โน้ตการประชุม หรือสายอีเมล ให้อัปโหลดได้เลย สกิลจะดึงคำพูดและลำดับความสำคัญที่สำคัญ

**ระบุกลุ่มผู้ชมให้ชัดเจน** — "ทีมเทคนิค" ดี แต่ "สถาปนิก IT ที่ประเมินโมเดลความปลอดภัย" ดีกว่า

**ปรับแต่งได้อย่างอิสระ** — ร่างแรกไม่ถูกใจ? เพียงบอกว่าต้องการเปลี่ยนอะไร สี ส่วน ข้อความ ลำดับงาน — ปรับได้ทั้งหมด

## คำถามที่พบบ่อย

**ถาม: ใช้ได้กับผลิตภัณฑ์/บริษัทอะไรก็ได้ไหม?**
ตอบ: ได้ สกิลจะตรวจจับว่าคุณขายอะไรและทำการวิจัยตามนั้น

**ถาม: รู้สีแบรนด์ของลูกค้าเป้าหมายได้อย่างไร?**
ตอบ: ดึงข้อมูลจาเว็บไซต์หรือแนวทางแบรนด์ของลูกค้าเป้าหมาย สามารถปรับเปลี่ยนได้ภายหลัง

**ถาม: ส่งออกเป็น PDF ได้ไหม?**
ตอบ: ได้ — ขอเวอร์ชันที่เหมาะสำหรับพิมพ์ แล้วใช้ฟังก์ชันพิมพ์เป็น PDF ของเบราว์เซอร์
