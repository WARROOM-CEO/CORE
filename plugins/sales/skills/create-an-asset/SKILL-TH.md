# สกิล: การสร้างสื่อการขาย

**สร้างสื่อการขายที่ปรับแต่งตามดีล — หน้า Landing Pages, พรีเซนเทชัน, แผ่นพับข้อมูล, การสาธิตกระบวนการทำงาน**

---

## สกิลนี้ทำอะไร?

สกิล `การสร้างสื่อการขาย` จะสร้างสื่อที่พร้อมแชร์กับลูกค้าโดยอิงจากบริบทของดีลจริง — วิจัยลูกค้าเป้าหมาย ถามคำถามสั้นๆ และสร้างสื่อที่มีแบรนด์และพร้อมใช้งานทันที

## รูปแบบที่รองรับ

| รูปแบบ | เหมาะสำหรับ |
|--------|------------|
| **หน้า Landing Page แบบ Interactive** | การประชุมผู้บริหาร การนำเสนอข้อเสนอคุณค่า — หลายแท็บพร้อมการสาธิตและเครื่องคำนวณ |
| **พรีเซนเทชัน** | การนำเสนอทางการ กลุ่มผู้ฟังขนาดใหญ่ — สไลด์แนวตั้งพร้อมระบบนำทาง |
| **แผ่นพับข้อมูล (One-Pager)** | เอกสารฝากไว้ สรุปอย่างรวดเร็ว — เลื่อนหน้าเดียว ข้อมูลครบถ้วน |
| **การสาธิตกระบวนการทำงาน** | การเจาะลึกทางเทคนิค POC — แผนภาพแบบ interactive พร้อมการเคลื่อนไหวแสดงลำดับงาน |

## กระบวนการทำงาน

```
1. คุณให้บริบท (ลูกค้าเป้าหมาย กลุ่มผู้ชม เป้าหมาย รูปแบบ)
        ↓
2. Claude วิจัยลูกค้าเป้าหมาย
        ↓
3. Claude ถาม 3-4 คำถามเพื่อยืนยันแนวทาง
        ↓
4. Claude สร้างสื่อการขาย
        ↓
5. คุณปรับแต่งตามต้องการ
```
